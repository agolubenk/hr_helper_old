# Политика конфиденциальности — расширение HR Helper

**Дата:** 2025  
**Расширение:** HR Helper (Chrome)

## 1. Назначение

Расширение HR Helper используется только вместе с системой HR Helper для автоматизации HR-процессов: интеграция с LinkedIn, Huntflow, календарём и Google Meet. Оно предназначено для внутреннего/корпоративного использования.

## 2. Какие данные использует расширение

- **Вкладки (tabs):** доступ к активной вкладке только на разрешённых сайтах (LinkedIn, Huntflow, Google Calendar, Google Meet) для отображения кнопок и передачи данных в HR Helper.
- **Хранилище (storage):** локальное сохранение настроек в браузере (URL сервера HR Helper, API-токен). Данные не покидают устройство пользователя кроме запросов к вашему серверу HR Helper.
- **Данные с страниц:** на страницах LinkedIn (профили, чаты) расширение может считывать публичные данные профиля и отправлять их только на ваш сервер HR Helper (например, ссылку на профиль, имя) для поиска кандидата или сохранения в ATS.

## 3. Передача данных третьим лицам

Расширение **не передаёт данные рекламным сетям и третьим сторонам**. Данные отправляются только на указанный пользователем сервер HR Helper (по умолчанию `https://hr.sftntx.com`) при наличии API-токена.

## 4. Разрешения (permissions)

| Разрешение      | Зачем нужно |
|-----------------|-------------|
| `storage`       | Сохранение настроек (URL, API-токен) в браузере. |
| `tabs`         | Определение активной вкладки для работы на LinkedIn/Huntflow. |
| `activeTab`    | Доступ к текущей вкладке по запросу пользователя. |
| `host_permissions` | Доступ к LinkedIn, Huntflow, Google Calendar/Meet и вашему серверу HR Helper для API-запросов и инъекции кнопок. |
