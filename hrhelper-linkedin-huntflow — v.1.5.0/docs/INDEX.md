## Дополнительные материалы

В папке `docs/old/` могут храниться устаревшие или справочные материалы (например старые гайды по установке, CORS). Актуальная документация — в корне `docs/`: README, USER_GUIDE, ARCHITECTURE, INDEX.

В корне проекта:

- **manifest.json** — конфигурация расширения (версия, разрешения, content scripts, иконки).
- **icons/** — иконки расширения (в т.ч. README по генерации).
- **shared/** — общие модули (constants, utils, api client), используемые в content scripts и при необходимости в других контекстах.
