# Исправления проблем с загрузкой данных

## ✅ Исправленные проблемы

### 1. Ошибка пагинации Huntflow API
**Проблема:** 
- API возвращал ошибку: `"ensure this value is greater than 0"` для параметра `page`
- Мы передавали `page=0` (0-based), но API требует `page >= 1` (1-based)

**Исправление:**
```python
# Было:
page=page - 1,  # Huntflow использует 0-based пагинацию

# Стало:
page=max(int(request.GET.get('page', 1)), 1)  # Huntflow использует 1-based пагинацию, минимум 1
```

### 2. Ошибка ограничения count
**Проблема:**
- API возвращал ошибку: `"ensure this value is less than or equal to 30"` для параметра `count`
- Мы передавали `count=50`, но API ограничивает до 30

**Исправление:**
```python
# Было:
count = int(request.GET.get('count', 50))

# Стало:
count = min(int(request.GET.get('count', 30)), 30)  # Huntflow API ограничивает count до 30
```

### 3. Обновлены опции в шаблоне
**Изменения:**
- Убраны опции `count=50` и `count=100`
- Оставлены только валидные значения: 10, 20, 30
