# Wiki App - Документация системы

## Описание

Приложение Wiki предоставляет систему документации для HR Helper. Позволяет создавать, редактировать и организовывать документацию по всем разделам системы с использованием тегов, категорий и связей между страницами.

## Возможности

- ✅ **Создание страниц** с Markdown контентом
- ✅ **Система тегов** для категоризации и поиска
- ✅ **Категории** для организации страниц
- ✅ **Связи с приложениями** для фильтрации
- ✅ **История изменений** для отслеживания правок
- ✅ **Поиск** по содержимому страниц
- ✅ **Фильтрация** по тегам, категориям и приложениям

## Структура

### Модели

- **WikiPage** — страницы документации
- **WikiTag** — теги для категоризации
- **WikiPageHistory** — история изменений страниц

### Категории

- **Введение** — вводные страницы
- **Архитектура** — архитектура системы
- **Настройка** — руководства по настройке
- **Использование** — руководства по использованию
- **Интеграции** — интеграции с внешними системами

### Теги

- `#настройка` — настройка системы и модулей
- `#использование` — руководства по использованию
- `#интеграции` — интеграции с внешними системами
- `#финансы` — финансовый модуль
- `#вакансии` — управление вакансиями
- `#календарь` — Google Calendar и инвайты
- `#ai` — AI-помощник
- `#интервьюеры` — управление интервьюерами
- `#метрики` — метрики и аналитика
- `#пользователи` — управление пользователями

## Использование

### Создание полной документации

Для создания полной документации Wiki со всеми тегами и страницами:

```bash
python manage.py create_complete_wiki
```

Эта команда:
1. Создает все необходимые теги
2. Обновляет существующие страницы, добавляя теги
3. Создает новые страницы для каждого приложения
4. Настраивает связи между страницами

### Создание базовых страниц

Для создания только базовых страниц:

```bash
python manage.py create_wiki_pages
```

### Доступ к Wiki

1. Перейдите в меню **Wiki**
2. Просмотрите список страниц
3. Используйте фильтры по тегам, категориям и приложениям
4. Используйте поиск для быстрого поиска

### Редактирование страниц

Только пользователи с правами `staff` могут редактировать страницы:

1. Перейдите на страницу Wiki
2. Нажмите кнопку **Редактировать**
3. Внесите изменения
4. Сохраните страницу

## Теги и хештеги

Все страницы помечены тегами для быстрой навигации. Используйте теги в тексте страниц для создания ссылок на связанные разделы.

Пример использования тегов в тексте:
```markdown
См. также [Настройка финансов](/wiki/page/finance-setup/) #финансы #настройка
```

## API

