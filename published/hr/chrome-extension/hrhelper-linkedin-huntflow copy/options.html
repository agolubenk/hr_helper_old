Настройки

По умолчанию: https://hr.sftntx.com
Получить токен: откройте в браузере (будучи залогиненным в HR Helper) страницу Интеграции — скопируйте API токен и вставьте в это поле. Можно вставить токен с префиксом «Token » или весь JSON ответ — расширение извлечёт ключ само.
Для работы расширения нужно быть залогиненным в HRHelper в этом же профиле Chrome (cookies).
