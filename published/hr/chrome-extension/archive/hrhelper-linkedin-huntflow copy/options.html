Настройки

По умолчанию: https://hr.sftntx.com
Получить токен: откройте в браузере (будучи залогиненным в HR Helper) /api/v1/accounts/users/token/ — скопируйте в это поле только значение из data.token (одна длинная строка), без префикса «Token » и без кавычек. Можно вставить и весь JSON ответ — расширение возьмёт ключ само.
Для работы расширения нужно быть залогиненным в HRHelper в этом же профиле Chrome (cookies).
