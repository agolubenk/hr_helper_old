# Установка расширения HR Helper (Chrome)

Расширение опубликовано в Chrome Web Store и доступно по прямой ссылке (в каталоге по поиску не отображается).

---

## Способ 1: Установка из Chrome Web Store (рекомендуется)

### 1. Откройте страницу расширения в магазине

Перейдите по ссылке:

**https://chromewebstore.google.com/detail/hr-helper/dccabghccldhpkichoejklfmbcclmepl**

(При необходимости добавьте параметры: `?authuser=0&hl=ru` — для языка интерфейса.)

### 2. Установите расширение

1. На странице расширения нажмите кнопку **«Установить»** (или «Add to Chrome» / «Добавить в Chrome»).
2. В диалоге подтверждения нажмите **«Установить расширение»**.
3. После установки иконка HR Helper появится на панели расширений Chrome (при необходимости закрепите её: нажмите на значок пазла → булавка рядом с «HR Helper»).

### 3. Получите API-токен

Токен нужен для доступа расширения к вашему серверу HR Helper.

1. Войдите в HR Helper в браузере под своей учётной записью.
2. Откройте в той же вкладке (или в новой) ссылку:
   - **https://hr.sftntx.com/api/v1/accounts/users/token/**  
   (если используете другой домен — замените `hr.sftntx.com` на ваш.)
3. На странице отобразится JSON с полем `"token"` и длинной строкой.
4. Скопируйте **только значение токена** (строку без кавычек).  
   Пример: если в ответе `"token": "abc123xyz..."`, копируйте `abc123xyz...`.
5. Сохраните токен — он понадобится в настройках расширения.

### 4. Настройте расширение

1. Нажмите на иконку HR Helper на панели Chrome (или: **chrome://extensions/** → найдите HR Helper → «Подробности» → «Настройки расширения» / ссылка на параметры).
2. В форме укажите:
   - **Base URL:** `https://hr.sftntx.com` (или ваш домен HR Helper).
   - **API Token:** вставьте скопированный токен.
3. Нажмите **«Сохранить»** (или аналог в форме).

### 5. Проверка работы

1. Откройте страницу профиля в **LinkedIn** (например, https://www.linkedin.com/in/...).
2. На странице должна появиться кнопка **Huntflow** (виджет расширения).
3. На странице резюме **hh.ru** или **rabota.by** при нажатии на иконку расширения откроется попап для привязки ссылки Huntflow.
4. На странице **Google Meet** (meet.google.com) при наличии связи встречи с кандидатом отобразятся кнопки расширения.
